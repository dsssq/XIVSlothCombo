* 快捷指令

  所有选项打钩
`/scombo setall`

  所有选项取消
`/scombo unsetall`

  逻辑取反
`/scombo toggle SimpleBardFeature`

  启用xx 功能
`/scombo set GunbreakerNoMercyonST`

  取消xx 功能
`/scombo unset SimpleBardFeature`  etc.
